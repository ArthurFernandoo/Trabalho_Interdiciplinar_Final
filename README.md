# Classificação de Tumores de Mama usando K-Nearest Neighbors (KNN)

Este repositório guarda os ficheiros do Trabalho Interdisciplinar do Grupo 4 (1º Semestre) - Dataset 9. O objetivo principal é programar e comparar o algoritmo de classificação K-Nearest Neighbors (KNN) de duas formas: feito do zero (manual) e usando uma biblioteca pronta.

## Ficheiros do Repositório
* `breast_cancer_dataset.csv`: Base de dados original (dados brutos).
* `1_analise_exploratoria.ipynb`: Notebook com a análise dos dados e a limpeza (pré-processamento).
* `2_implementacao_knn_e_comparacao.ipynb`: Notebook principal com o código manual, o código usando a biblioteca Scikit-Learn e a comparação dos resultados.
* `RELATÓRIO FINAL.pdf`: Relatório escrito explicando a escolha da base de dados, o que foi feito no código e a discussão sobre a ética do uso de IA na saúde.

## Como executar o projeto
1. Verifique se tem as bibliotecas `pandas`, `numpy` e `scikit-learn` instaladas no seu Python.
2. Execute primeiro o notebook `1_analise_exploratoria.ipynb` para limpar os dados e criar o ficheiro final.
3. Depois, execute o notebook `2_implementacao_knn_e_comparacao.ipynb` para ver o modelo a funcionar e conferir as previsões.
